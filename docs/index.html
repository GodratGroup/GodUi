Setup

To use GodUi, you need to add it to your project and setup it in the first step...

Import

To add GodUi, you can use two methods: Full Import and Advanced Import.

Full Import

In this method you will add GodUi to your project in full using the following code.

@import url(https://raw.githubusercontent.com/GodratGroup/GodUi/main/src/css/GodUi.css);

Advanced Import

In this way you will import only the items from GodUi that you need.

Config

You can easily adjust the primary and secondary color in GodUi.

Follow the code below

:root {
    --primary-color: [primary color hex, rgb & ...]
    --secondary-color: [secondary color hex, rgb & ...]
    --bg-primary-color: [background primary color hex, rgb & ...]
    --bg-secondary-color: [background secondary color hex, rgb & ...]
}
